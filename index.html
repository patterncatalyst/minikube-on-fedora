What you'll learn

How to install minikube on Fedora 44, run a local Kubernetes cluster with the podman driver, and deploy applications with both kubectl and helm. Later sections cover Istio service mesh and (optionally) KEDA HTTP-driven autoscaling — reference material for using those tools against a local cluster. All container images are UBI-based and do not require Red Hat subscription registration.
